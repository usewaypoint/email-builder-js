# Email builder with MUI integrated into a Vite application

Use this as a sample to self-host EmailBuilder.js inside of Vite

To run this locally, fork the repository and then run vite from within this directory:

```
cd examples/vite-emailbuilder-mui
npm install
npx vite
```

Once the server is running, open http://localhost:5173/email-builder-js/ in your browser.
